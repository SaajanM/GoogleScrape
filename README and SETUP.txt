Setup
------------------
Open a terminal in this directory
Run the following in the opened terminal:
  pip3 install -r requirements.txt
Then copy the google_scrape python file into your working directory (ie. the directory with your main python file)
Open your main file and add the following to your import statements:
  import google_scrape


Documentation
------------------
google_scrape.getNumResults(query)
     query = A string representation of your query
     returns: a number representing how many search results google had
